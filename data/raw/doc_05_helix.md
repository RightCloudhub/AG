# Helix Compute Overview

Helix Compute is an enterprise hardware company. Helix Compute competes with NovaTech Industries.

Helix Compute produces the HelixCore Server. SiliconForge supplies processors for HelixCore Server as well as for QuantumEdge Server.

The CEO of Helix Compute is Priya Nair. Priya Nair previously worked at Meridian Capital.
