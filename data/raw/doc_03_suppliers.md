# Supply Chain Notes: QuantumEdge Server

The QuantumEdge Server product depends on several key suppliers.

SiliconForge supplies advanced chips for QuantumEdge Server. Harbor Components supplies power modules for QuantumEdge Server.

Interestingly, SiliconForge also supplies chips for Helix Compute's flagship servers. Harbor Components supplies only NovaTech Industries in the enterprise server segment.

BrightLink Logistics provides shipping services to both NovaTech Industries and Helix Compute distribution centers.
